GitBook安装

安装Nodejs

  • 下载
  • 安装
  • 验证
    node -v
    打印出版本信息

安装Gitbook命令

  • 使用npm命令安装，输入

    npm install -g gitbook-cli

  • 然后再输入命令：

    gitbook -V

    注意，是大写的V，这里可能会执行升级操作，等待完成即可。
